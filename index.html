BSON - Binary JSON

			Bson:
			  \x16\x00\x00\x00           // total document size
  			  \x02                       // 0x02 = type String
  			  hello\x00                  // field name
  			  \x06\x00\x00\x00world\x00  // field value
			                             // (size of value, value)
			  \x00                       // 0x00 = type EOO
			                             // ('end of object')
